## 🎮 Como Usar

### 1. Configuração do Webhook
1. Abra o site no seu navegador
2. Insira a URL do webhook do Discord no campo apropriado
3. Clique em "Testar" para verificar a conexão

### 2. Configuração do Raid
- **Mensagem**: Digite a mensagem que será enviada
- **Quantidade**: Defina quantas mensagens enviar (1-100)
- **Delay**: Tempo entre mensagens em milissegundos (100-10000)
- **Nome de Usuário**: Nome personalizado para o bot (opcional)
- **Avatar URL**: URL do avatar personalizado (opcional)
- **Randomizar**: Ative para variar as mensagens automaticamente

### 3. Executar o Raid
1. Configure todos os parâmetros desejados
2. Clique em "Iniciar Raid" ou pressione Ctrl+Enter
3. Monitore o progresso na seção de controles
4. Use "Parar Raid" ou Esc para interromper

## ⌨️ Atalhos do Teclado

- `Ctrl + Enter`: Iniciar raid
- `Esc`: Parar raid
- `Ctrl + L`: Limpar logs (quando implementado)
